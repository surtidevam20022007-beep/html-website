# Simple Portfolio Website

This is a simple portfolio website created using **HTML and CSS**.
It contains a main page and a second page connected with a button.

## 📁 Project Files

* `index.html` – Main homepage of the portfolio
* `page2.html` – Second page opened using a button
* `style.css` – CSS file used for styling both pages

## 🌐 Features

* Simple and clean design
* Button that opens another page
* External CSS styling
* Beginner-friendly project

## 🛠 Technologies Used

* HTML
* CSS

## 🚀 How to Run

1. Download or clone the repository.
2. Make sure all files are in the same folder.
3. Open `index.html` in your web browser.

## 📷 Preview
